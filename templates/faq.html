⚙️ Sobre o Software

O que é o NFE Reader?

O NFE Reader é um software para Windows que lê arquivos XML de Notas Fiscais Eletrônicas (NF-e) e extrai automaticamente todos os dados para planilhas CSV/Excel. Ele gera 10 planilhas base + 4 relatórios consolidados (contas a pagar, ICMS recuperável, etc.), eliminando a digitação manual.

Preciso pagar para testar?

Não. Você tem 30 dias grátis com todas as funcionalidades liberadas, sem necessidade de cartão de crédito.

Funciona no Mac ou Linux?

Não. Atualmente o NFE Reader é compatível apenas com Windows 7, 8, 10 e 11 (64 bits).

Preciso de internet para usar?

Só para ativar a licença e renovar. O processamento dos XMLs é 100% offline — seus dados fiscais não saem do seu computador.

Quais arquivos o programa aceita?

Arquivos XML de NF-e (modelo 55) e NFC-e (modelo 65). O programa também lê XMLs dentro de subpastas e arquivos .ZIP.

Posso processar XMLs de qualquer empresa?

Sim. O programa apenas lê os dados do XML — não há restrição de CNPJ emitente ou destinatário.

🔐 Licenciamento

O que é "licença por equipamento"?

Cada licença do NFE Reader é vinculada ao hardware específico do computador (HWID). Isso significa que a chave de ativação de uma máquina não funciona em outra. É uma medida de segurança para evitar compartilhamento indevido.

A chave funciona em outro computador?

Não. Cada licença é única por equipamento. Se você trocar de computador, precisará de uma nova chave para a máquina nova.

Formatei o PC. Perdi minha licença?

Não. A licença continua válida para o mesmo hardware. Basta reinstalar o NFE Reader, registrar com o mesmo CPF/CNPJ e contatar o suporte para reativar sua chave.

Quanto tempo dura cada plano?

Mensal: 30 dias  |  Trimestral: 90 dias  |  Anual: 365 dias  |  Vitalício: 36.500 dias (tecnicamente vitalício)

Posso renovar antes do vencimento?

Sim, mas a nova chave substitui a anterior — os dias restantes da licença antiga são perdidos. Recomendamos renovar próximo ao vencimento.

Formas de pagamento aceitas?

Atualmente aceitamos apenas PIX, processado pela plataforma Asaas. O pagamento é confirmado em segundos e a chave é gerada automaticamente.

Como funciona o vitalício? São atualizações gratuitas?

O plano vitalício dá direito ao uso do software por tempo indeterminado. Correções de bugs são gratuitas para todos os planos. Novas funcionalidades podem ser cobradas à parte, mesmo para vitalícios.

📥 Instalação e Uso

Como faço para instalar?

1. Acesse nfe-licensing-server.onrender.com
2. Clique em "Download"
3. Execute o arquivo NFE_Reader.exe baixado
4. Preencha seus dados (nome, CPF/CNPJ, e-mail) e clique em "Registrar e Começar"

O Windows acusou "proteção" ao executar. É seguro?

Sim, é seguro. Como o software é novo, ainda não possui certificado digital de editor (assinatura). Isso é normal para softwares independentes. Clique em "Executar assim mesmo" — o código é aberto e verificável no repositório oficial.

Como processar minhas primeiras notas?

1. Clique em "🔍 Selecionar" na pasta de entrada e escolha onde estão seus XMLs
2. Clique em "🔍 Selecionar" na pasta de saída e escolha onde salvar os CSVs
3. Clique no botão verde "🚀 PROCESSAR XMLs"
4. Após processar, clique em "📄 EXPORTAR CSV" para gerar as planilhas

O que significa "XML Bloqueado"?

O programa bloqueia o processamento de XMLs apenas quando a licença está expirada. Com a licença ativa, todos os XMLs são processados normalmente. Nenhum dado é perdido — ao renovar, você volta a processar normalmente.

O que são os relatórios consolidados?

São relatórios prontos que combinam dados de várias categorias:
contas_a_pagar.csv — tudo que vence, venceu ou vai vencer
icms_recuperavel.csv — ICMS a recuperar nas compras
consolidado_notas.csv — resumo financeiro de todas as notas
consolidado_itens.csv — cada produto com impostos detalhados

Como abrir os CSVs no Excel?

Basta clicar duas vezes no arquivo. O CSV já vem no formato brasileiro: separador ; e vírgula para decimais. Se abrir bagunçado, no Excel vá em: Dados → De Texto/CSV → Escolher separador ponto e vírgula.

O programa lê XMLs já processados de novo?

Não. Após processar, os XMLs são movidos para uma subpasta "Processados". Na próxima execução, apenas XMLs novos (ainda não processados) serão lidos. Isso evita retrabalho e duplicidade.

🔒 Dados e Privacidade

O NFE Reader envia minhas notas fiscais para algum servidor?

Não. O processamento é 100% local. Os XMLs são lidos e processados no seu computador. Nenhum dado fiscal é enviado para servidores externos. A única comunicação online é para ativar/renovar a licença.

Quais dados meus são armazenados?

Apenas os dados do registro: nome, CPF/CNPJ, e-mail e HWID (identificador do hardware). Consulte nossa Política de Privacidade para detalhes completos.

Posso solicitar a exclusão dos meus dados?

Sim. Entre em contato pelo e-mail suporte.onucleo@gmail.com solicitando a exclusão, conforme seus direitos garantidos pela LGPD. Seus dados serão removidos, exceto quando a lei exigir manutenção (ex.: obrigações fiscais).

💳 Pagamento e Reembolso

O pagamento é seguro?

Sim. O pagamento é processado inteiramente pela plataforma Asaas, uma das maiores fintechs do Brasil, registrada no Banco Central. Seus dados bancários não passam pelo nosso servidor.

Quanto tempo leva para a chave ser gerada após o pagamento?

O PIX é confirmado em segundos. A chave de licença é gerada automaticamente assim que o webhook confirma o pagamento. Na maioria dos casos, o processo leva menos de 30 segundos.

Posso pedir reembolso?

Licenças vitalícias têm garantia de 7 dias para reembolso (direito de arrependimento - art. 49 CDC). Para licenças de período fixo (mensal, trimestral, anual), não há reembolso após a ativação da chave, pois o produto digital foi integralmente entregue.

Perdi a chave de ativação. O que faço?

Entre em contato pelo e-mail suporte.onucleo@gmail.com informando seu CPF/CNPJ. Podemos recuperar sua chave ou gerar uma nova para o mesmo equipamento.
